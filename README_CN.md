# 全球大学列表

一个全面的全球大学数据库，为每个院校提供标准化的**英文**和**中文**名称。

[English](./README.md)

## 项目概览

本项目维护了一个精心筛选的全球高等教育机构列表。作为一个中心化数据库，每个大学条目包含：
- **_id**: 唯一标识符。
- **中文名称**: 官方简体中文名称。
- **英文名称**: 标准化国际英文名称（通过 AI 翻译/验证）。
- **国家**: 中英文对照的国家名称。

**总记录数:** 约 10,300+ 条，覆盖 100+ 个国家和地区。

## 🚀 快速开始

整个项目由一个统一的脚本 `main.py` 管理。

### 环境准备
1.  Python 3.8+
2.  安装依赖:
    ```bash
    pip install pandas google-genai python-dotenv
    ```
3.  在 `scripts/.env` 文件中设置您的 Gemini API Key (用于翻译服务):
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```

### 使用说明
运行主管理脚本来规范化数据、翻译缺失名称并生成最终汇总表：

```bash
python3 main.py
```

该脚本执行以下三项核心任务：
1.  **规范化 (Normalize)**: 标准化 CSV 表头并清理每个国家文件夹中的格式。
2.  **翻译 (Translate)**: 扫描非英文名称（如法语、西班牙语、俄语等），并调用 **Gemini 2.0 Flash** 将其翻译为标准英文。
3.  **汇总 (Summarize)**: 将所有国家的数据聚合到主文件 `world_universities.csv` 中。

## 📂 项目结构

数据按国家分类存储在 `data/` 目录下，配置文件位于 `scripts/` 目录。

```
.
├── main.py                     # 核心管理脚本
├── world_universities.csv      # 最终输出 (主数据库)
├── README.md                   # 英文说明
├── README_CN.md                # 中文说明
├── data/                       # 原始数据目录
│   ├── China/                  # 中国数据（包含抓取脚本）
│   ├── USA/                    # 美国数据
│   └── ...                     # 其他国家
└── scripts/                    # 配置文件和脚本工具
    ├── .env                    # 环境变量
    ├── ssl_adapter.py          # 网络适配器
    └── openssl_legacy.cnf      # SSL 配置
```

## ✨ 核心特性

- **自动化 AI 翻译**: 利用 Gemini API 识别非英文校名（如 "Université de Paris" 或 "東京大学"），并将其翻译为官方英文名称。
- **严格英文校验**: 强制最终输出使用“仅限英文”的名称。自动识别并翻译诸如 *Universidad*, *Ecole*, *Hochschule* 等词汇。
- **数据标准化**: 自动清理表头、移除重复列，并修正常见的格式问题（如引号错误、多余空格）。
- **特殊地区处理**: 对**香港**、**澳门**和**台湾**进行特殊标记，确保在全球列表中格式正确。

## 数据来源
- **中国**: 通过特定脚本抓取（保留在 `data/China/` 文件夹中）。
- **全球**: 汇总自各官方列表，并标准化为统一格式。
